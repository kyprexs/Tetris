## üì¶ Getting Started

1. **Clone the repository:**
   ```sh

   cd tetris
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

   ```sh
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
